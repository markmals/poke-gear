# PokéGear

Guides and tools for Pokémon games, built on top of [Astro](https://astro.build)'s [Starlight](https://starlight.astro.build).

| Command       | Action                                      |
| :------------ | :------------------------------------------ |
| `bun install` | Installs dependencies                       |
| `bun dev`     | Starts local dev server at `localhost:4321` |
